BAB 1

A. Menggunakan array
Konsep array digunakan secara luas karena memberikan banyak kemudahan kepada programmer ketika menuliskan kode program dan menyelesaikan persoalan dengan pendekatan yang lebih mudah. Array banyak digunakan terutama ketika bekerja dengan perhitungan matematis atau ketika bekerja dengan perulangan. Array adalah sekumpulan variabel yang memiliki tipe data yang sama dan disimpan di lokasi memori yang berdekatan dan mempunyai nama (identifier) yang sama.

B. Memberi Nilai Pada sebuah Array
Suatu array secara default tidak memiliki nilai (null). Ini berarti elemen-elemen yang dimiliki array tersebut juga tidak mempunyai nilai. Ketika menambahkan nilai ke array maka perlu diperhatikan bahwa jumlah nilai yang ada di dalam tanda kurung (), tidak boleh lebih banyak dari jumlah elemen array yang dideklarasikan.

C. Mengakses Nilai-nilai sebuah Array
Nilai-nilai yang ada di sebuah array dapat diakses dengan cara seperti mengakses variabel. Hal yang perlu dilakukan adalah memberikan secara spesifik nilai array dari elemen yang ingin diakses. Ketika mengakses sebuah nilai array, kita cukup memanggil nama array tersebut dan menentukan dengan spesifik indeks dari elemen array yang ingin diakses.

D. Menggunakan Multi-dimensi Array
Array yang digunakan untuk menyimpan (hold) rangkaian nilai yang telah disusun dalam suatu baris, dapat dibayangkan nilai-nilai tersebut ditempatkan di dalam suatu baris memori. Array multidimensi adalah array yang terdiri dari kumpulan array atau sering disebut array dari array (arrays of arrays) multi_array adalah nama dari array multidimensi Sedangkan baris dan kolom adalah jumlah baris dan kolom dari array yang dimiliki.

E. Menggunakan Array dalam Perhitungan
Bagaimana cara menggunakan array dalam sebuah persamaan matematika, mendeklarasikan array dan memberi nilai, mendeklarasikan array tanpa memberi nilai, memberikan nilai ke array dan kemudian menggunakan elemen array dalam perhitungan matematika.
Langkah-langkah untuk melakukannya adalah sebagai berikut:

1. Jalankan aplikasi wxDev C++
2. Dari menu File. Pilihlah menu New dan klik perintah Source File. Jendela edit untuk menuliskan kode program akan ditampilkan
3. Tambahkan kode program yang ditunjukkan oleh Gambar
4. Dari menu Execute, jalankan perintah Compile. Jika masih ada kesalahan, perbaiki kesalahan tersebut dan jalankan kembali perintah Compile.
5. Jika tidak ada lagi kesalahan, dari menu Execute jalankan perintah Run without Debugging. Program akan dijalankan
6. Perhatikan tampilan program, apakah bekerja sesuai dengan yang diharapkan.

Menggunakan Array Multidimensi

Nilai matriks akan disimpan dalam bentuk array multidimensi dan dengan menggunakan aturan perkalian matriks di atas, akan dilakukan perkalian pada elemen-elemen array yang ada. Langkah-langkah untuk membuat program perkalian matriks menggunakan array multidimensi adalah berikut:

1. Jalankan aplikasi wxDev C++.

2. Dari menu File, pilihlah menu New dan klik perintah Source File. Jendela Edit untuk menuliskan kode program akan ditampilkan.

3. Tambahkan kode program

4. Dari menu Execute, jalankan perintah Compile. Jika masih ada kesalahan, perbaiki kesalahan tersebut dan jalankan kembali perintah Compile.

5. Jika tidak ada lagi kesalahan, dari menu Execute, jalankan perintah Run without Debugging. Program akan dijalankan.

Mendefinisikan Fungsi Sendiri

Fungsi sebenamya adalah kumpulan statemen atau proses yang bekerja untuk menyelesaikan sebuah tugas. Jika ingin menambahkan dan menggunakan fungsi lain, maka fungsi yang lain tersebut adalah sebuah fungsitambahan

a. Return Type Return Type adalah tipe data dari nilai yang dikembalikan oleh fungsi tersebut

b. Nama Fungsi Nama fungsi adalah nama dari fungsi. Nama tersebut akan digunakan sebagai identifier ketika akan memanggil fungsi tersebut

c. Daftar Parameter Daftar parameter merupakan sebuah tempat untuk menempatkan nilai-nilai dari luar untuk diproses di dalam fungsi.

d. Bagian Utama Fungsi Bagian utama fungsi terdiri dari kumpulan statemen yang akan dijalankan ketiga fungsi dipanggil kadang kala jika membutuhkan fungsi yang spesifik, fungsi tidak tersedia pada kesempatan ini akan dibahas bagaimana membuat fungsi yang didefinisikan sendiri dan bagaimana memanggilnya dari sebuah kode program.
